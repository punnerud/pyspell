Join your own tailnet — a browser node 🧪

This runs our Rust tailscale-core as WebAssembly (≈129 kB) so this browser tab becomes a node on your own Tailscale tailnet — control + DERP over WebSocket, no app, no hardware. Click Login, open the URL it gives you in another window to authorize this node, and it'll come online. Your node keys stay in this browser's localStorage (a refresh reuses the same node and the same login URL — no new one each time). This is a Phase-0 spike: it proves registration + gets the IP.
